# Sources Template

The master list of sources the Monday 9am task pulls from each week.

**Update protocol:** in chat, tell Claude *"add [X] to sources"* or *"remove [X]"* and it updates this file plus the Monday task prompt.

---

## Tier 1 — Primary (weight 40%, target 2 of 5 posts)

### Your own first-person content

The single highest-trust source: your own videos / blog / podcast / live talks. Two of the 5 weekly ideas should anchor on something you've recently published.

- **YouTube channel:** [https://www.youtube.com/@YOURHANDLE]
- **Blog (optional):** [URL]
- **Podcast (optional):** [URL]

The Monday task fetches recent content and uses it to ground posts like *"In this week's video I walked through X — here's the takeaway"* or *"Coming out of my latest YouTube about Y — three things that didn't make the cut."*

## How the Monday task uses this

1. Reads this file
2. Reads `03-ad-hoc-finds.md` (your weekly scratchpad)
3. Web-searches Tier 2 sources for fresh angles this week
4. Fetches your own channel for recent content (Tier 1)
5. Generates 5 idea candidates mixing the tiers above
6. Surfaces an interactive review widget (Approve / Edit / Skip / Cancel)
